选项 用户选择 系统分析 A 当场站起来说"这个方案其实是我做的" 🔴 情绪驱动型。你看到了不公，但你的第一反应是"纠正"，而不是"谋利"。洞察力✅ 分辨力❌。触发偏差：零和博弈思维——你觉得必须赢回来，但忽略了"当众撕破脸"的长期代价 B 会后私下找领导说"这个方案我也参与了部分" 🟡 低调谋利型。你选择了成本最低的方式拿回功劳，但用词是"参与"而不是"主导"，说明你在保护关系的同时在争取利益。洞察力✅ 分辨力✅。思维标签：务实派 C 什么都不说，但开始偷偷给领导发邮件同步自己的工作进展 🟢 长期布局型。你不争一时，而是建立"可见度"。这是最高级的职场策略——让领导自己发现真相。洞察力✅✅ 分辨力✅✅。思维标签：系统思维者 测的是：利益冲突时，你是被情绪带着走，还是能冷静计算最优解？ 关联偏差：零和博弈思维 / 情绪劫持 / 短视偏差

场景 02 | 「站队时刻」 公司两个副总明争暗斗，现在都来拉拢你。A总许诺给你升职，B总说会带你去新项目。你选哪个？

选项 用户选择 系统分析 A 选A总，因为升职是确定的 🔴 锚定效应。你被"升职"这个确定收益锚定了，忽略了A总可能明年就被架空。你看到了眼前利益，但没看到局势。洞察力❌ 分辨力❌ B 选B总，因为新项目有想象空间 🟡 乐观偏差。你看到了可能性，但"想象空间"是模糊的，你其实是在用感觉做决策，不是用分析。洞察力⚠️ 分辨力⚠️ C 谁都不选，先观察3个月再决定 🟢 延迟决策+信息收集型。你意识到现在信息不够，不轻易下注。这是最成熟的策略。洞察力✅✅ 分辨力✅✅。思维标签：耐心猎人 测的是：面对不确定性，你能不能忍住不选？ 关联偏差：锚定效应 / 乐观偏差 / 沉没成本（如果你已经被A总拉过一次）

场景 03 | 「领导的烂摊子」 领导让你接手一个明显要黄的项目，但他说"做好了年底给你加薪"。你知道这个项目大概率失败，你接不接？

选项 用户选择 系统分析 A 接，领导的话不能不听 🔴 权威服从+短期利益驱动。你把"领导说的"当事实，忽略了"大概率失败"这个信号。分辨力❌。触发偏差：权威偏误 B 不接，直接拒绝 🟡 回避风险型。你看到了风险，但你的反应是"逃"，不是"谈条件"。你缺少谈判思维。洞察力✅ 分辨力⚠️ C 接，但提前跟领导谈好："如果项目失败，我要什么补偿/资源" 🟢 风险对冲型。你不是不接，也不是盲接，而是把一个必输的局变成了"输了也不亏"的局。洞察力✅✅ 分辨力✅✅。思维标签：博弈高手 测的是：你面对"明知是坑"的任务，是服从/逃避/还是重新设计规则？ 关联偏差：权威偏误 / 损失厌恶 / 框架效应

场景 04 | 「同事的背刺」 你跟同事关系不错，有天你无意中发现他在领导面前说你坏话。你怎么办？

选项 用户选择 系统分析 A 找他对质："你为什么在背后说我？" 🔴 情绪宣泄型。你需要一个答案，但对质只会让他更防备你，而且你永远得不到真话。洞察力⚠️ 分辨力❌。触发偏差：基本归因错误——你觉得他是"坏人"，而不是分析他为什么这么做 B 以后防着他，不再跟他说任何事 🟡 防御型。你看到了威胁，但你的策略是"隔离"，这会让你在团队里越来越孤立。洞察力✅ 分辨力⚠️ C 不动声色，但开始在领导面前展示自己的价值，同时对他保持表面友好 🟢 以攻为守型。你明白：在职场，最好的反击不是撕破脸，而是让领导看到谁更有价值。洞察力✅✅ 分辨力✅✅。思维标签：暗棋选手 测的是：被背叛后，你是被情绪控制，还是能把坏事变成好事？ 关联偏差：基本归因错误 / 情绪劫持 / 短期思维

📂 类别二：金钱决策（4个） 场景 05 | 「朋友借钱」 关系很好的朋友找你借5万，说3个月还。但你知道他上个月刚换了新车，信用卡还欠着。你借不借？

选项 用户选择 系统分析 A 借，朋友嘛，不能不帮 🔴 关系绑架+乐观偏差。你把"关系好"当成了"他会还"的证据。你看到了他换车，但你选择性忽略了。分辨力❌。触发偏差：可得性启发——你想到的是"他以前对我好" B 不借，直接说没钱 🟡 保守型。你看到了风险，但你的判断是基于"他肯定不会还"，这其实也是一种偏见（也许他真的有急用）。洞察力⚠️ 分辨力⚠️ C 借2万，跟他说"我也紧张，只能帮这么多，你写个借条" 🟢 风险可控型。你既维护了关系，又控制了风险，还留了证据。你能在"情义"和"理性"之间找到平衡点。洞察力✅✅ 分辨力✅✅。思维标签：成熟决策者 测的是：金钱+感情混在一起时，你能不能分开看？ 关联偏差：可得性启发 / 乐观偏差 / 关系绑架

场景 06 | 「打折陷阱」 你本来只想买一双500块的鞋，但商家说"满1000减300"，你最后花了980买了两双鞋。第二双你其实不需要。你怎么看这件事？

选项 用户选择 系统分析 A "我赚了啊，省了300呢" 🔴 框架效应+锚定效应。你被"省了300"这个框架控制了，但你实际多花了480。你看到了"折扣"，没看到"多花的钱"。分辨力❌ B "我被套路了，以后再也不买了" 🟡 过度矫正。你意识到了问题，但反应是"一刀切"，这说明你对自己的判断力不自信。洞察力✅ 分辨力⚠️ C "这次确实多花了，但我学到了——以后买东西先算'实际单价'而不是'省了多少'" 🟢 元认知型。你不只是后悔，你还从这次错误中提取了一个可复用的决策规则。洞察力✅✅ 分辨力✅✅。思维标签：学习型大脑 测的是：你犯了错之后，是只会后悔，还是能提炼出规则？ 关联偏差：框架效应 / 锚定效应 / 损失厌恶（"不买就亏了"）

场景 07 | 「投资诱惑」 有人给你推荐一个"稳赚不赔"的项目，年化收益30%，已经有200人投了。你投不投？

选项 用户选择 系统分析 A 投，这么多人都投了应该靠谱 🔴 从众效应+幸存者偏差。你看到的200人是"还在的人"，亏了跑了的你看不到。你用"人多"当证据，而不是用逻辑分析。分辨力❌ B 不投，这种都是骗子 🟡 过度怀疑。你可能是对的，但你的判断是基于"感觉"而不是"分析"。也许这个项目真的有逻辑，你只是懒得想。洞察力⚠️ C 不投，但我会去查这个项目的底层逻辑——钱从哪来？利润从哪来？如果说不清，那就是庞氏骗局 🟢 结构化质疑型。你不是盲目信，也不是盲目不信，你用了一个分析框架去验证。洞察力✅✅ 分辨力✅✅。思维标签：批判性思维者 测的是：面对高收益诱惑，你是跟风/排斥/还是用框架验证？ 关联偏差：从众效应 / 幸存者偏差 / 贪婪+恐惧

场景 08 | 「沉没成本」 你花2000块买了一张演唱会门票，但当天你发烧了。去的话很难受，不去的话票就废了。你怎么选？

选项 用户选择 系统分析 A 去，票都买了不能浪费 🔴 沉没成本谬误。你做决策的依据是"已经花了2000"，而不是"去了值不值"。2000已经花了，不管你去不去都回不来。分辨力❌ B 不去，在家休息 🟡 理性但缺乏灵活性。你做对了，但你可能忽略了一个变量：有没有朋友可以转票？你只看到了"去/不去"两个选项。洞察力⚠️ C 不去，但挂闲鱼看看能不能转掉，或者送给朋友，至少回本一部分 🟢 选项扩展型。你不仅跳出了沉没成本，还主动创造了第三个选项。这是最高级的决策方式。洞察力✅✅ 分辨力✅✅。思维标签：创造性解决者 测的是：你能不能不被"已经花了的钱/时间"绑架？ 关联偏差：沉没成本谬误 / 二元思维（只看到两个选项）

📂 类别三：关系冲突（4个） 场景 09 | 「伴侣的谎言」 你发现伴侣偷偷给前任转了500块。他说是"她有急用，借的"。你信不信？

选项 用户选择 系统分析 A 不信，直接摊牌："你是不是还想着她？" 🔴 情绪驱动+确认偏误。你已经预设了"他在骗我"，然后去找证据支持这个结论。不管他说什么你都不会信。分辨力❌。触发偏差：确认偏误 B 信了，但心里一直不舒服 🟡 压抑型。你选择了相信，但你没有真正处理这个情绪。你在用"忍"代替"解决"。洞察力⚠️ 分辨力⚠️ C 不当场发作，但之后观察他的行为模式——如果类似的事再发生一次，就认真谈；如果没有，这次就当偶然事件 🟢 概率思维+分层决策型。你不是"信或不信"的二元选择，而是给了一个"观察期+触发条件"。这是最成熟的关系处理方式。洞察力✅✅ 分辨力✅✅。思维标签：理性温柔者 测的是：在关系中，你是被情绪控制，还是能冷静分层处理？ 关联偏差：确认偏误 / 情绪劫持 / 二元思维

场景 10 | 「父母的控制」 父母坚决反对你的职业选择，说"你这样会饿死的"。但你自己做了调研，这个方向3年后市场很大。你怎么办？

选项 用户选择 系统分析 A 听父母的，他们不会害我 🔴 权威依赖+短期安全感驱动。你把"父母说的"等同于"对的"，但你忽略了一个事实：父母的经验来自20年前的市场。分辨力❌。触发偏差：权威偏误 + 现状偏差 B 不管他们，我行我素 🟡 叛逆但不独立。你反抗了，但你的反抗是"因为他们反对所以我偏要做"，这不是独立思考，这是情绪化对抗。洞察力⚠️ C 认真做一份数据报告给父母看，用他们能理解的方式说明"为什么这件事能成"，给他们3个月观察期 🟢 说服型独立。你既坚持了自己的判断，又尊重了父母的担忧，还用"数据"而不是"感觉"去沟通。洞察力✅✅ 分辨力✅✅。思维标签：成熟独立者 测的是：面对至亲的反对，你是盲从/对抗/还是用他们能接受的方式坚持自己？ 关联偏差：权威偏误 / 现状偏差 / 情绪化决策

场景 11 | 「朋友的越界」 你最好的朋友总喜欢拿你的缺点开玩笑，每次你说"别这样"，他都说"开个玩笑而已，你太敏感了"。你怎么办？

选项 用户选择 系统分析 A 算了，他可能真的没恶意 🔴 自我合理化+讨好型。你把"他的越界"合理化成"他没恶意"，本质上是你不想失去这段关系。你看到了问题，但你选择了忍受。分辨力❌ B 爆发："你再这样我们就别做朋友了" 🟡 情绪爆发型。你终于说了，但方式是威胁，这会让对方防御而不是反思。你的意图是好的，但方法错了。洞察力✅ 分辨力⚠️ C 认真地、平静地说一次："这个玩笑让我不舒服，我需要你停下来。如果你做不到，我需要重新考虑我们的关系。"说完就看他的行动，不再多说 🟢 边界清晰型。你没有攻击，没有忍耐，而是设立了清晰的边界+后果。这是最健康的关系处理方式。洞察力✅✅ 分辨力✅✅。思维标签：边界感强者 测的是：你能不能在不伤害关系的前提下，保护自己的边界？ 关联偏差：自我合理化 / 讨好型人格 / 情绪化表达

场景 12 | 「社交压力」 朋友们都在买房，你爸妈也在催，但你算了一下，买房后你每个月要还8000，生活质量会大幅下降。你买不买？

选项 用户选择 系统分析 A 买，大家都买了我不买显得我混得差 🔴 社会比较+从众效应。你的决策依据不是"我需不需要"，而是"别人怎么看我"。你被外部评价绑架了。分辨力❌。触发偏差：社会证明 + 锚定效应（别人的房价成了你的锚） B 不买，谁爱买谁买 🟡 对抗但不理性。你抵抗了压力，但你的理由是"不想跟风"，而不是"我算过账了不划算"。你在用情绪对抗情绪。洞察力⚠️ C 不买，但我会认真算一笔账：如果我不买房，这笔钱投到哪里回报率更高？然后用数据跟父母和自己交代 🟢 独立计算型。你不是"不买"，你是"算清楚了再决定"。你能在社会压力下保持独立计算能力。洞察力✅✅ 分辨力✅✅。思维标签：理性独立者 测的是：当所有人都在做同一件事时，你能不能停下来自己算一笔账？ 关联偏差：从众效应 / 社会比较 / 锚定效应

📂 类别四：自我认知（4个） 场景 13 | 「连续失败」 你连续3次面试都没过。第4次机会来了，你的第一反应是？

场景 14 | 「被夸的时候」 有人夸你"你真聪明"，你第一反应是？

场景 15 | 「选择困难」 两个工作机会：A稳定但无聊，B有挑战但可能失败。你纠结了一周了，还没决定。你怎么看这个状态？
