🟡 核心挑战在哪里？
1️⃣ 科学性质疑（最大风险）
"你说我洞察力差，凭什么？你的标准是什么？"

如果只是主观评价 → 用户觉得是"巴纳姆效应"（说谁都对）
如果没有底层框架支撑 → 容易被当成"玄学算命"
解决方案：需要绑定一个可信的认知框架（比如双系统理论、认知偏差清单、博弈论思维等）
2️⃣ 竞品已经很多了
竞品	你和它们的区别必须是？
MBTI / 16Personalities	静态标签 vs 你的动态场景模拟 ← 这是你的壁垒
盖洛普优势识别	测优势 vs 你测思维盲区 ← 角度不同
BetterUp / 各种教练平台	真人贵 vs 你AI+结构化 ← 价格优势
ChatGPT直接问	没有结构、没有评估体系、没有成长路径 ← 你的价值
3️⃣ 复购率可能很低
测一次 → "哦我是这样的人" → 然后呢？→ 流失
必须设计成长闭环：测评 → 训练 → 再测 → 对比进步
4️⃣ 伦理和合规
涉及"人格判断"容易踩线（尤其在中国）
不能暗示"你有问题"，要包装成"你有提升空间"
不能替代心理咨询（要有免责声明）
🔵 如果要做，建议的产品架构
📌 定位：不是"测试你是什么人"，而是"训练你怎么看问题"

第1层：场景模拟（入口/引流）
  → 给一个真实人生困境（职场/感情/投资）
  → 用户做选择 → 系统分析你的思维路径
  → 揭示你的认知偏差（如：锚定效应、幸存者偏差...）

  → 不是打分，而是生成"思维画像"
  → 比如：你在利益冲突场景下容易情绪驱动（分辨力↓）
  →       你在长期规划上容易忽略隐性变量（洞察力↓）

第3层：训练模式（留存/复购）
  → 每周推送新场景
  → 追踪你的决策质量是否在提升
  → 类似"思维健身房"

## 写出第一批20个场景的具体内容？直接能用的那种

📂 类别一：职场博弈（4个）
场景 01 | 「功劳被抢」
你熬了两周做的方案，在会上被同事当众汇报了，领导还夸他"做得不错"。你知道真相，但没有证据。你怎么做？

选项	用户选择	系统分析
A	当场站起来说"这个方案其实是我做的"	🔴 情绪驱动型。你看到了不公，但你的第一反应是"纠正"，而不是"谋利"。洞察力✅ 分辨力❌。触发偏差：零和博弈思维——你觉得必须赢回来，但忽略了"当众撕破脸"的长期代价
B	会后私下找领导说"这个方案我也参与了部分"	🟡 低调谋利型。你选择了成本最低的方式拿回功劳，但用词是"参与"而不是"主导"，说明你在保护关系的同时在争取利益。洞察力✅ 分辨力✅。思维标签：务实派
C	什么都不说，但开始偷偷给领导发邮件同步自己的工作进展	🟢 长期布局型。你不争一时，而是建立"可见度"。这是最高级的职场策略——让领导自己发现真相。洞察力✅✅ 分辨力✅✅。思维标签：系统思维者
测的是：利益冲突时，你是被情绪带着走，还是能冷静计算最优解？
关联偏差：零和博弈思维 / 情绪劫持 / 短视偏差

场景 02 | 「站队时刻」
公司两个副总明争暗斗，现在都来拉拢你。A总许诺给你升职，B总说会带你去新项目。你选哪个？

选项	用户选择	系统分析
A	选A总，因为升职是确定的	🔴 锚定效应。你被"升职"这个确定收益锚定了，忽略了A总可能明年就被架空。你看到了眼前利益，但没看到局势。洞察力❌ 分辨力❌
B	选B总，因为新项目有想象空间	🟡 乐观偏差。你看到了可能性，但"想象空间"是模糊的，你其实是在用感觉做决策，不是用分析。洞察力⚠️ 分辨力⚠️
C	谁都不选，先观察3个月再决定	🟢 延迟决策+信息收集型。你意识到现在信息不够，不轻易下注。这是最成熟的策略。洞察力✅✅ 分辨力✅✅。思维标签：耐心猎人
测的是：面对不确定性，你能不能忍住不选？
关联偏差：锚定效应 / 乐观偏差 / 沉没成本（如果你已经被A总拉过一次）

场景 03 | 「领导的烂摊子」
领导让你接手一个明显要黄的项目，但他说"做好了年底给你加薪"。你知道这个项目大概率失败，你接不接？

选项	用户选择	系统分析
A	接，领导的话不能不听	🔴 权威服从+短期利益驱动。你把"领导说的"当事实，忽略了"大概率失败"这个信号。分辨力❌。触发偏差：权威偏误
B	不接，直接拒绝	🟡 回避风险型。你看到了风险，但你的反应是"逃"，不是"谈条件"。你缺少谈判思维。洞察力✅ 分辨力⚠️
C	接，但提前跟领导谈好："如果项目失败，我要什么补偿/资源"	🟢 风险对冲型。你不是不接，也不是盲接，而是把一个必输的局变成了"输了也不亏"的局。洞察力✅✅ 分辨力✅✅。思维标签：博弈高手
测的是：你面对"明知是坑"的任务，是服从/逃避/还是重新设计规则？
关联偏差：权威偏误 / 损失厌恶 / 框架效应

场景 04 | 「同事的背刺」
你跟同事关系不错，有天你无意中发现他在领导面前说你坏话。你怎么办？

选项	用户选择	系统分析
A	找他对质："你为什么在背后说我？"	🔴 情绪宣泄型。你需要一个答案，但对质只会让他更防备你，而且你永远得不到真话。洞察力⚠️ 分辨力❌。触发偏差：基本归因错误——你觉得他是"坏人"，而不是分析他为什么这么做
B	以后防着他，不再跟他说任何事	🟡 防御型。你看到了威胁，但你的策略是"隔离"，这会让你在团队里越来越孤立。洞察力✅ 分辨力⚠️
C	不动声色，但开始在领导面前展示自己的价值，同时对他保持表面友好	🟢 以攻为守型。你明白：在职场，最好的反击不是撕破脸，而是让领导看到谁更有价值。洞察力✅✅ 分辨力✅✅。思维标签：暗棋选手
测的是：被背叛后，你是被情绪控制，还是能把坏事变成好事？
关联偏差：基本归因错误 / 情绪劫持 / 短期思维

A	借，朋友嘛，不能不帮	🔴 关系绑架+乐观偏差。你把"关系好"当成了"他会还"的证据。你看到了他换车，但你选择性忽略了。分辨力❌。触发偏差：可得性启发——你想到的是"他以前对我好"
B	不借，直接说没钱	🟡 保守型。你看到了风险，但你的判断是基于"他肯定不会还"，这其实也是一种偏见（也许他真的有急用）。洞察力⚠️ 分辨力⚠️
C	借2万，跟他说"我也紧张，只能帮这么多，你写个借条"	🟢 风险可控型。你既维护了关系，又控制了风险，还留了证据。你能在"情义"和"理性"之间找到平衡点。洞察力✅✅ 分辨力✅✅。思维标签：成熟决策者

关联偏差：可得性启发 / 乐观偏差 / 关系绑架

场景 06 | 「打折陷阱」
你本来只想买一双500块的鞋，但商家说"满1000减300"，你最后花了980买了两双鞋。第二双你其实不需要。你怎么看这件事？

选项	用户选择	系统分析
A	"我赚了啊，省了300呢"	🔴 框架效应+锚定效应。你被"省了300"这个框架控制了，但你实际多花了480。你看到了"折扣"，没看到"多花的钱"。分辨力❌
B	"我被套路了，以后再也不买了"	🟡 过度矫正。你意识到了问题，但反应是"一刀切"，这说明你对自己的判断力不自信。洞察力✅ 分辨力⚠️
C	"这次确实多花了，但我学到了——以后买东西先算'实际单价'而不是'省了多少'"	🟢 元认知型。你不只是后悔，你还从这次错误中提取了一个可复用的决策规则。洞察力✅✅ 分辨力✅✅。思维标签：学习型大脑
测的是：你犯了错之后，是只会后悔，还是能提炼出规则？
关联偏差：框架效应 / 锚定效应 / 损失厌恶（"不买就亏了"）

场景 07 | 「投资诱惑」
有人给你推荐一个"稳赚不赔"的项目，年化收益30%，已经有200人投了。你投不投？

选项	用户选择	系统分析
A	投，这么多人都投了应该靠谱	🔴 从众效应+幸存者偏差。你看到的200人是"还在的人"，亏了跑了的你看不到。你用"人多"当证据，而不是用逻辑分析。分辨力❌
B	不投，这种都是骗子	🟡 过度怀疑。你可能是对的，但你的判断是基于"感觉"而不是"分析"。也许这个项目真的有逻辑，你只是懒得想。洞察力⚠️
C	不投，但我会去查这个项目的底层逻辑——钱从哪来？利润从哪来？如果说不清，那就是庞氏骗局	🟢 结构化质疑型。你不是盲目信，也不是盲目不信，你用了一个分析框架去验证。洞察力✅✅ 分辨力✅✅。思维标签：批判性思维者
测的是：面对高收益诱惑，你是跟风/排斥/还是用框架验证？
关联偏差：从众效应 / 幸存者偏差 / 贪婪+恐惧

场景 08 | 「沉没成本」
你花2000块买了一张演唱会门票，但当天你发烧了。去的话很难受，不去的话票就废了。你怎么选？

选项	用户选择	系统分析
A	去，票都买了不能浪费	🔴 沉没成本谬误。你做决策的依据是"已经花了2000"，而不是"去了值不值"。2000已经花了，不管你去不去都回不来。分辨力❌
B	不去，在家休息	🟡 理性但缺乏灵活性。你做对了，但你可能忽略了一个变量：有没有朋友可以转票？你只看到了"去/不去"两个选项。洞察力⚠️
C	不去，但挂闲鱼看看能不能转掉，或者送给朋友，至少回本一部分	🟢 选项扩展型。你不仅跳出了沉没成本，还主动创造了第三个选项。这是最高级的决策方式。洞察力✅✅ 分辨力✅✅。思维标签：创造性解决者
测的是：你能不能不被"已经花了的钱/时间"绑架？
关联偏差：沉没成本谬误 / 二元思维（只看到两个选项）

📂 类别三：关系冲突（4个）
场景 09 | 「伴侣的谎言」
你发现伴侣偷偷给前任转了500块。他说是"她有急用，借的"。你信不信？

选项	用户选择	系统分析
A	不信，直接摊牌："你是不是还想着她？"	🔴 情绪驱动+确认偏误。你已经预设了"他在骗我"，然后去找证据支持这个结论。不管他说什么你都不会信。分辨力❌。触发偏差：确认偏误
B	信了，但心里一直不舒服	🟡 压抑型。你选择了相信，但你没有真正处理这个情绪。你在用"忍"代替"解决"。洞察力⚠️ 分辨力⚠️
C	不当场发作，但之后观察他的行为模式——如果类似的事再发生一次，就认真谈；如果没有，这次就当偶然事件	🟢 概率思维+分层决策型。你不是"信或不信"的二元选择，而是给了一个"观察期+触发条件"。这是最成熟的关系处理方式。洞察力✅✅ 分辨力✅✅。思维标签：理性温柔者

场景 10 | 「父母的控制」
父母坚决反对你的职业选择，说"你这样会饿死的"。但你自己做了调研，这个方向3年后市场很大。你怎么办？

选项	用户选择	系统分析
A	听父母的，他们不会害我	🔴 权威依赖+短期安全感驱动。你把"父母说的"等同于"对的"，但你忽略了一个事实：父母的经验来自20年前的市场。分辨力❌。触发偏差：权威偏误 + 现状偏差
B	不管他们，我行我素	🟡 叛逆但不独立。你反抗了，但你的反抗是"因为他们反对所以我偏要做"，这不是独立思考，这是情绪化对抗。洞察力⚠️
C	认真做一份数据报告给父母看，用他们能理解的方式说明"为什么这件事能成"，给他们3个月观察期	🟢 说服型独立。你既坚持了自己的判断，又尊重了父母的担忧，还用"数据"而不是"感觉"去沟通。洞察力✅✅ 分辨力✅✅。思维标签：成熟独立者
测的是：面对至亲的反对，你是盲从/对抗/还是用他们能接受的方式坚持自己？
关联偏差：权威偏误 / 现状偏差 / 情绪化决策

场景 11 | 「朋友的越界」
你最好的朋友总喜欢拿你的缺点开玩笑，每次你说"别这样"，他都说"开个玩笑而已，你太敏感了"。你怎么办？

选项	用户选择	系统分析
A	算了，他可能真的没恶意	🔴 自我合理化+讨好型。你把"他的越界"合理化成"他没恶意"，本质上是你不想失去这段关系。你看到了问题，但你选择了忍受。分辨力❌
B	爆发："你再这样我们就别做朋友了"	🟡 情绪爆发型。你终于说了，但方式是威胁，这会让对方防御而不是反思。你的意图是好的，但方法错了。洞察力✅ 分辨力⚠️
C	认真地、平静地说一次："这个玩笑让我不舒服，我需要你停下来。如果你做不到，我需要重新考虑我们的关系。"说完就看他的行动，不再多说	🟢 边界清晰型。你没有攻击，没有忍耐，而是设立了清晰的边界+后果。这是最健康的关系处理方式。洞察力✅✅ 分辨力✅✅。思维标签：边界感强者
测的是：你能不能在不伤害关系的前提下，保护自己的边界？
关联偏差：自我合理化 / 讨好型人格 / 情绪化表达

场景 12 | 「社交压力」
朋友们都在买房，你爸妈也在催，但你算了一下，买房后你每个月要还8000，生活质量会大幅下降。你买不买？

选项	用户选择	系统分析
A	买，大家都买了我不买显得我混得差	🔴 社会比较+从众效应。你的决策依据不是"我需不需要"，而是"别人怎么看我"。你被外部评价绑架了。分辨力❌。触发偏差：社会证明 + 锚定效应（别人的房价成了你的锚）
B	不买，谁爱买谁买	🟡 对抗但不理性。你抵抗了压力，但你的理由是"不想跟风"，而不是"我算过账了不划算"。你在用情绪对抗情绪。洞察力⚠️
C	不买，但我会认真算一笔账：如果我不买房，这笔钱投到哪里回报率更高？然后用数据跟父母和自己交代	🟢 独立计算型。你不是"不买"，你是"算清楚了再决定"。你能在社会压力下保持独立计算能力。洞察力✅✅ 分辨力✅✅。思维标签：理性独立者
测的是：当所有人都在做同一件事时，你能不能停下来自己算一笔账？
关联偏差：从众效应 / 社会比较 / 锚定效应

📂 类别四：自我认知（4个）
场景 13 | 「连续失败」

选项	用户选择	系统分析

B	"这次肯定能过！"	🟡 盲目乐观。你在用"希望"代替"分析"。你没有从前3次失败中学到任何东西。洞察力❌

场景 14 | 「被夸的时候」

选项	用户选择	系统分析
A	开心，觉得自己确实聪明	🔴 固定型思维。你把"聪明"当成了一个固定标签，这会让你害怕挑战（因为挑战可能证明你"不聪明"）。洞察力⚠️

C	谢谢，然后想"我在哪件事上做对了？下次可以复制"	🟢 成长型思维+元认知型。你把夸奖转化成了"可复用的行为"。这是最高级的自我认知方式。洞察力✅✅ 分辨力✅✅。思维标签：元认知者

场景 15 | 「选择困难」
两个工作机会：A稳定但无聊，B有挑战但可能失败。你纠结了一周了，还没决定。你怎么看这个状态？

选项	用户选择	系统分析
A	"我就是选择困难症，没办法"	🔴 标签化自我。你给自己贴了个标签，然后用这个标签当借口不做决策。你回避了真正的问题：你到底在怕什么？分辨力❌
B	"选A吧，稳定最重要"	🟡 安全优先但没想清楚。你做了选择，但你的理由是"怕失败"，不是"我真正想要什么"。洞察力⚠️
C	"我之所以纠结，是因为我两个都想要——既要稳定又要挑战。我需要先想清楚：我现在人生阶段最需要的是什么？"	🟢 问题重构型。你没有纠结"选A还是B"，而是把问题从"选哪个"变成了"我现在需要什么"。这是真正的洞察力。洞察力✅✅ 分辨力✅✅。思维标签：问题重构者
测的是：你面对纠结时，是给自己贴标签/随便选一个/还是重新定义问题？
关联偏差：选择过载 / 标签化思维 / 框架效应

场景 16 | 「别人的成功」
你刷到同龄人买了房/升了职/结了婚，你心里什么感觉？

选项	用户选择	系统分析
A	焦虑，"我是不是落后了"	🔴 社会比较+稀缺心态。你把别人的成功当成了"你的失败"。你的参照系是别人，不是自己。分辨力❌。触发偏差：社会比较 + 零和思维
B	无所谓，关我什么事	🟡 情感隔离。你不是真的无所谓，你是不敢面对那个焦虑。你在用"不在乎"保护自己。洞察力⚠️
C	"他的路径不一定适合我。我的节奏是什么？我现在最该做的是什么？"	🟢 参照系切换型。你把参照系从"别人"切换到了"自己"。你能在信息洪流中保持自己的节奏。洞察力✅✅ 分辨力✅✅。思维标签：内驱型人格
测的是：你的自我评价体系是建在"和别人比"上，还是"和自己比"上？
关联偏差：社会比较 / 稀缺心态 / 锚定效应（别人的成功成了你的锚）

场景 17 | 「90%的赌局」
一个投资：90%概率赚10万，10%概率亏50万。你投不投？

选项	用户选择	系统分析
A	投！90%呢，稳赚	🔴 概率忽视+乐观偏差。你看到了90%，但你忽略了那10%的后果是"亏50万"。你在用"概率高"代替"期望值计算"。期望值 = 0.9×10 - 0.1×50 = 9-5 = 4万，是正的，但你的决策方式不对。分辨力❌
B	不投，万一亏了呢	🟡 损失厌恶主导。你不是在算概率，你是在怕亏。你把"可能亏50万"的恐惧放大了10倍。洞察力⚠️
C	算一下：我能承受最大亏损是多少？如果50万亏了我不会死，那从期望值看可以投；如果会影响生活，那不投	🟢 风险承受力+期望值双维度型。你没有被概率骗，也没有被恐惧吓，你用了一个完整的决策框架。洞察力✅✅ 分辨力✅✅。思维标签：量化决策者
测的是：面对概率，你是看数字/怕损失/还是用框架算？
关联偏差：概率忽视 / 损失厌恶 / 乐观偏差

场景 18 | 「黑天鹅」
你的生活一直很顺，突然公司裁员，你被裁了。你的第一反应？

选项	用户选择	系统分析
A	"完了，天塌了"	🔴 灾难化思维。你把一个事件等同于"人生完蛋了"。你看不到这件事之外的任何可能性。分辨力❌。触发偏差：灾难化 + 隧道视野
B	"早知道就不该在这家公司待这么久"	🟡 后视偏误。你在用"已经知道的结果"去否定"当初的选择"。但当初你做决定时，信息是不一样的。洞察力⚠️
C	"虽然突然，但我有存款能撑3个月。先更新简历，同时想想有没有其他可能"	🟢 韧性+行动导向型。你接受了现实，但没有被现实打倒。你的第一反应不是情绪，而是"我现在能做什么"。洞察力✅✅ 分辨力✅✅。思维标签：反脆弱者
测的是：意外来临时，你是崩溃/后悔/还是立刻进入"解决模式"？
关联偏差：灾难化思维 / 后视偏误 / 隧道视野

场景 19 | 「信息爆炸」
你想做一个副业，网上搜了一圈：有人说做自媒体能月入10万，有人说自媒体已经死了，有人说应该做跨境电商……你怎么办？

选项	用户选择	系统分析
A	选那个看起来最赚钱的	🔴 幸存者偏差+锚定效应。你看到的"月入10万"是活下来的人，死了的你看不到。你被最大的数字锚定了。分辨力❌
B	算了，太乱了不做了	🟡 信息过载导致决策瘫痪。你不是不想做，你是被信息量压垮了。你缺少"信息筛选框架"。洞察力⚠️
C	"这些信息里，哪些是有数据支撑的？哪些是个人感受？哪些是卖课的？我只看有数据的，然后找3个真实在做的人私下聊"	🟢 信息分级+一手验证型。你能在信息洪流中建立筛选标准，并且知道"网上说的不算，要找真人验证"。洞察力✅✅ 分辨力✅✅。思维标签：信息猎手
测的是：面对矛盾信息，你是跟风最亮的/放弃/还是建立筛选框架？
关联偏差：幸存者偏差 / 信息过载 / 权威偏误（大V说的=对的？）

场景 20 | 「未来的自己」
如果10年后的你穿越回来，看到你现在的生活状态，你觉得他会说什么？

选项	用户选择	系统分析
A	"你怎么还在做这个？赶紧换啊！"	🔴 用未来焦虑绑架现在。你在用一个想象中的"更好的自己"来否定现在的自己。但你没想过：也许现在的选择就是当时最优的。分辨力❌
B	"还行吧，至少没饿死"	🟡 低期待+自我安慰。你不是真的在思考，你是在给自己找台阶。你回避了真正的问题。洞察力⚠️
C	"他可能会说：你现在做的事，方向对不对？如果方向对，慢一点没关系；如果方向错了，现在改还来得及"	🟢 方向性思维型。你不纠结"快不快"，你在想"对不对"。这是最成熟的人生决策框架。洞察力✅✅ 分辨力✅✅。思维标签：长期主义者
测的是：你做决策时，看的是"速度"还是"方向"？
关联偏差：时间贴现 / 现状偏差 / 规划谬误

核心逻辑：

✅ 高级留存：让他觉得"不续费 = 我之前的钱白花了"

不是"留住用户"，是让他自己不想走。

📐 全局架构：3个阶段 × 8种用户

退款率高的核心原因只有一个：他觉得"也就那样"
所以我们要做的是：在他觉得"也就那样"之前，让他看到变化

每一个节点，都是一个"他不想走"的理由。

不是打分。不是分类。

六、定价
方案	价格	说明
免费	0元	每天1次急诊机会
单次急诊	9.9元	无限次，当天有效
急诊包（5次）	19.9元	5张卡，随时用
主推19.9的急诊包。

为什么不推9.9单次？

因为9.9单次 → 用户用1次就走。
19.9五次 → 用户会存着，下次还要用。
而且19.9/5 = 3.98元/次，用户觉得"一次才4块钱"，比9.9便宜太多。

实际上客单价从9.9变成了19.9。

七、增长模型
用户A：纠结要不要借钱给朋友
    → 输入 → 拿到急诊卡 → "关系绑架"
    → 觉得准 → 发给朋友B："你也测测，你肯定也是关系绑架"
    → 朋友B：测了 → 果然准 → 买了19.9急诊包
    → 朋友B：下次纠结 → 又来用
    → 循环
裂变驱动力不是"图好看"，是"这卡真准你也测测"。

这比"图好看"强10倍。

因为用户不是在帮你推广。
是在帮朋友避坑。
